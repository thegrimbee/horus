How do I locate the folder?

User Guide

To start using Horus, select the main folder your app is located in. Then click on the "Scan" button to start the scanning process.

Your app's main folder will most likely be in the "Program Files" or "Program Files (x86)" folder.

If you are using a Mac, it will be in the "Applications" folder.
